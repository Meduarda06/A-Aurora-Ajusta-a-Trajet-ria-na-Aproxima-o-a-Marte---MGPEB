# 🚀 MGPEB Aurora Siger - Gerenciamento Pouso Marte


**Módulo Gerenciamento Pouso e Estabilização Base (MGPEB)** para missão colônia Marte. Implementa filas, portas lógicas, funções matemáticas e governança ESG.

## 📋 Explicação do Projeto

**Atividade Integradora:** Simula pouso 10 módulos orbitais na base Aurora Siger (Marte).

### **Funcionalidades Implementadas:**
- 📦 **Estruturas Lineares:** Queue (fila principal) + Listas (pousados/alerta)
- 🔧 **Portas Lógicas:** AND/OR/NOT (combustível ∧ prioridade ∧ atmosfera)
- 🔍 **Algoritmos:** Busca linear + Bubble Sort prioridade
- 📈 **Função Matemática:** h(t) = -½gt² + v₀t + h₀ (pouso)
- 🏛️ **Governança ESG:** Prioridade médica > logística
- 💾 **Export:** Excel 4 abas + CSVs

**Resultado:** `5/10 módulos pousados com sucesso (50%)`
